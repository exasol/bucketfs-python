class exasol_bucketfs_utils_python.abstract_bucketfs_location.AbstractBucketFSLocation¶

Bases: abc.ABC

Abstract class for a BucketFSLocation for uploading and downloading strings, fileobjects and joblib objects. Also able to read files from the BucketFS directly, if called from inside a UDF.
